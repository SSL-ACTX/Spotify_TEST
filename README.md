## Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/Spotify_TEST.git
   cd Spotify_TEST
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Run the application:

   ```bash
   npm start
   ```

4. Open your browser and navigate to [http://localhost:5500](http://localhost:5500).
